# Martian Robots

Sequential navigation of Robots on the rectangular surface of Mars

### Tech Stack
- SpringBoot2
- Java


### Step to run 

Just a simple Springboot app that run using default profile
```shell
$ mvn clean install -U
$ mvn spring-boot:run
```
